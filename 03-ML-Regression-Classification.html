Supervised Learning: Regression vs Classification

Regression และ Classification เป็นงานหลักสองประเภทในการเรียนรู้แบบมีผู้สอน (Supervised Learning) ซึ่งมีความแตกต่างกันในเป้าหมายและผลลัพธ์ที่ต้องการ

Regression (การถดถอย)

ทำนายค่าต่อเนื่อง (Continuous Value) เช่น ราคา, อุณหภูมิ, น้ำหนัก

Classification (การจำแนกประเภท)

ทำนายคลาสหรือหมวดหมู่ (Discrete Class) เช่น สแปม/ไม่สแปม, ประเภทดอกไม้, การวินิจฉัยโรค

เปรียบเทียบ Regression และ Classification

คุณลักษณะ Regression Classification
ผลลัพธ์ ค่าต่อเนื่อง (Continuous Value) คลาสหรือหมวดหมู่ (Discrete Class)
ตัวอย่างผลลัพธ์ ราคาบ้าน: 3.5 ล้านบาท, อุณหภูมิ: 28.5°C ประเภทอีเมล: สแปม/ไม่สแปม, ประเภทดอกไม้: กุหลาบ/ทิวลิป/ลิลลี่
การวัดประสิทธิภาพ MSE, RMSE, MAE, R-squared Accuracy, Precision, Recall, F1-score, ROC-AUC
ฟังก์ชันการสูญเสีย (Loss Function) Mean Squared Error, Mean Absolute Error Cross-Entropy Loss, Hinge Loss
ตัวอย่างการใช้งาน ทำนายราคาบ้าน, ทำนายยอดขาย, ทำนายอุณหภูมิ จำแนกอีเมลสแปม, วินิจฉัยโรค, จำแนกรูปภาพ

อัลกอริทึมสำหรับ Regression

อัลกอริทึมที่นิยมใช้สำหรับงาน Regression เพื่อทำนายค่าต่อเนื่อง

Linear Regression

หาความสัมพันธ์เชิงเส้นระหว่างตัวแปรอิสระและตัวแปรตาม

สมการ: Y = β₀ + β₁X₁ + β₂X₂ + ... + βₙXₙ + ε

Polynomial Regression

ขยายจาก Linear Regression โดยใช้ความสัมพันธ์แบบพหุนาม

สมการ: Y = β₀ + β₁X + β₂X² + β₃X³ + ... + ε

Ridge Regression

Linear Regression ที่มีการเพิ่ม L2 regularization เพื่อลดปัญหา Overfitting

สมการ: min(||Y - Xβ||² + α||β||²)

อัลกอริทึมสำหรับ Classification

อัลกอริทึมที่นิยมใช้สำหรับงาน Classification เพื่อทำนายคลาสหรือหมวดหมู่

Logistic Regression

ใช้ฟังก์ชัน Sigmoid เพื่อทำนายความน่าจะเป็นที่ข้อมูลจะอยู่ในคลาสใดคลาสหนึ่ง

สมการ: P(Y=1) = 1 / (1 + e^(-z))

Decision Trees

สร้างโมเดลการตัดสินใจในรูปแบบของต้นไม้ โดยแต่ละโหนดแทนการตัดสินใจบนคุณลักษณะหนึ่งๆ

เกณฑ์การแบ่ง: Gini Impurity, Information Gain

Support Vector Machines

หาเส้นแบ่ง (Hyperplane) ที่ดีที่สุดเพื่อแยกข้อมูลออกเป็นคลาสต่างๆ

เทคนิค: Linear, Polynomial, RBF Kernel

อัลกอริทึมที่สามารถทำได้ทั้ง Regression และ Classification

อัลกอริทึมบางตัวมีความยืดหยุ่นและสามารถปรับใช้ได้ทั้งงาน Regression และ Classification

Decision Trees

Regression

ใช้ค่าเฉลี่ยของข้อมูลในโหนดใบ (Leaf Node) เป็นค่าทำนาย

Classification

ใช้คลาสที่พบมากที่สุดในโหนดใบเป็นคลาสที่ทำนาย

ไลบรารี: sklearn.tree.DecisionTreeRegressor, sklearn.tree.DecisionTreeClassifier

Random Forest

Regression

ใช้ค่าเฉลี่ยของผลลัพธ์จากต้นไม้ตัดสินใจหลายๆ ต้น

Classification

ใช้การโหวตเสียงข้างมากจากต้นไม้ตัดสินใจหลายๆ ต้น

ไลบรารี: sklearn.ensemble.RandomForestRegressor, sklearn.ensemble.RandomForestClassifier

Neural Networks

Regression

ใช้ฟังก์ชันกระตุ้น (Activation Function) แบบเชิงเส้นที่เอาต์พุต

Classification

ใช้ฟังก์ชันกระตุ้นแบบ Sigmoid หรือ Softmax ที่เอาต์พุต

ไลบรารี: sklearn.neural_network.MLPRegressor, sklearn.neural_network.MLPClassifier

Support Vector Machines

Regression (SVR)

หาฟังก์ชันที่มีค่าความคลาดเคลื่อนไม่เกิน ε จากค่าจริง

Classification (SVC)

หาเส้นแบ่งที่ดีที่สุดเพื่อแยกข้อมูลออกเป็นคลาสต่างๆ

ไลบรารี: sklearn.svm.SVR, sklearn.svm.SVC

เลือกประเภทของปัญหาด้านบนเพื่อดูตัวอย่าง

สรุป

เมื่อใดควรใช้ Regression

  • เมื่อต้องการทำนายค่าต่อเนื่อง เช่น ราคา, อุณหภูมิ, น้ำหนัก
  • เมื่อต้องการหาความสัมพันธ์ระหว่างตัวแปรอิสระและตัวแปรตาม
  • เมื่อต้องการวิเคราะห์แนวโน้มของข้อมูล
  • ตัวอย่างการใช้งาน: ทำนายราคาบ้าน, ทำนายยอดขาย, ทำนายอุณหภูมิ

เมื่อใดควรใช้ Classification

  • เมื่อต้องการจำแนกข้อมูลออกเป็นคลาสหรือหมวดหมู่
  • เมื่อผลลัพธ์ที่ต้องการเป็นค่าไม่ต่อเนื่อง (Discrete)
  • เมื่อต้องการตัดสินใจว่าข้อมูลเป็นประเภทใด
  • ตัวอย่างการใช้งาน: จำแนกอีเมลสแปม, วินิจฉัยโรค, จำแนกรูปภาพ
